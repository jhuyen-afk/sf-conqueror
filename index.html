First Time Setup:

  1. Enable Mac Permissions:
    • Open System Preferences
    • Go to Security & Privacy > Privacy
    • Enable Accessibility permissions
    • Allow Chrome automation
  2. Get Your Salesforce URL:
    • Log into Salesforce
    • Go to Leads tab
    • Click "New"
    • Select "US Sales"
    • Click "Next"
    • Copy the entire URL from your browser
    This URL contains your unique Salesforce ID which is needed for automation. Each user needs their own URL for the automation to work correctly.
  3. Tell Goose you're new:
    JHSF Conqueror
    Goose will guide you through:
    • Setting up your personal URL
    • Creating your own custom command
    • Testing the automation
